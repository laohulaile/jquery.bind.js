# 插件介绍

jquery.bind.js这个是实用的[jQuery](http://jquery.com/)插件，在一些html结构上绑定特定的自定义属性，那么，他将有相对应的js特效。

# 开始使用
在html页面中添加jquery和bind.js
```html
<script src="jquery-1.11.2.min.js"></script>
<script src="bind.js"></script>
```
